# Ethical Use Note

This software is provided as a tool for organizing thoughts and discussions
in a calm, structural manner.

It is not designed for:
- attacking individuals
- silencing others
- winning arguments through pressure, fear, or intimidation

The purpose of this tool is to:
- preserve ideas without emotional distortion
- make disagreements visible without escalation
- allow reflection after discussions have ended

If you intend to use this software to dominate, harass, or intimidate others,
this project is not aligned with that purpose.

This note is not a legal restriction.
It is a statement of intent.

---

## 倫理的利用について（日本語）

このソフトウェアは、思考や議論を  
**落ち着いた構造として整理するための道具**です。

以下の目的での利用は想定していません。

- 特定の個人を攻撃すること  
- 他者を黙らせること  
- 威圧や恐怖によって議論に勝つこと  

このツールの目的は、

- 感情による歪みを減らし  
- 意見の違いを可視化し  
- 話し合いのあとで考え直せる余地を残すこと  

です。

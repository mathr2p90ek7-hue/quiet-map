# quiet map

A small tool for keeping thoughts calm and visible.

This tool helps you record discussions and ideas **as structure, not emotion**.

Loud voices, anger, or pressure do not gain persuasive power here.
What remains are claims, reasons, counterpoints, and shifts of perspective.

quiet map is not for winning arguments.
It is a place to think **after** discussions have ended.

---

## quiet map について（日本語）

このツールは、議論や話し合いの内容を  
**感情ではなく構造として残す**ための、小さなアプリです。

大きな声、怒り、威圧は、  
ここでは説得力を持ちません。

残るのは、  
主張・理由・反論・論点のずれだけです。

quiet map は、  
その場で勝つためのものではありません。  
**あとから冷静に考えるための場所**を提供します。

---

## Basic Usage

1. Write down one thought or opinion as a node.
2. Add reasons or counterpoints and connect them with arrows.
3. If the topic shifts, place it separately without forcing a connection.
4. Review the whole map or export it as text.

There is no need to reach a conclusion.
Leaving disagreements visible is also a valid outcome.

---

## 使い方（要点）

1. 思ったことを一つ、ノードとして書きます  
2. 理由や反対意見を、矢印でつなぎます  
3. 論点がずれたら、無理につなげず別枠に置きます  
4. 全体を眺めたり、文章として出力します  

結論が出なくても構いません。  
意見が分かれたまま残ることも、大切な結果です。

---

## What this tool does NOT do

- It does not decide who is right.
- It does not force consensus.
- It does not reward speed or dominance.
- It does not judge emotions.

This tool exists to preserve thoughts, not to control people.

---

## このツールがしないこと

- 正解を決めません  
- 合意を強制しません  
- 声の大きさを評価しません  
- 感情を裁きません  

考えたことが壊れずに残ること。  
それだけを目的にしています。
